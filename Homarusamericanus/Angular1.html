LESSON 1 : Angular requires the following Script:

<script src= "http://ajax.googleapis.com/ajax/libs/angularjs/1.2.26/angular.min.js"></script>

DIRECTIVES 1: First Directive

add the
"ng-app"
DIRECTIVE to the tag where you want angular to Appear/function. EXAMPLE: <body ng-app>

DIRECTIVES 2: Create A model from Html Input

Add the
ng-model="name"
Directive to any Input tag or any HTML control that accepts a value. EXAMPLE: <p>Name: <input type="text" ng-model="name"></p>

DIRECTIVES 3: Bind your model data

Add the
ng-bind="name"
Directive to any tag. EXAMPLE:<p ng-bind="name"></p>

DIRECTIVES 4: Initialize your model data

Add
ng-init="firstName='John'"
Directive to any tag

EXPRESSIONS 1: Are written with {{ }}{{ expression }} and can be used instead of ng-bind="expression"

enclose everything in mustaches.

EXPRESSIONS 2: A Division Math Problem:

843 / 42 = {{ 843 / 42 }}

DIRECTIVES 5: Include partial pages

Add the
ng-include src="'/path/to/YourPartial.html'"
to a Div tag
