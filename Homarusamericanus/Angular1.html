DIRECTIVES 1: First Directive

add the "ng-app" DIRECTIVE to the tag where you want angular to Appear/function. EXAMPLE: <body ng-app>

DIRECTIVES 2: Create A model from Html Input

Add the ng-model="name" Directive to any Input tag or any HTML control that accepts a value. EXAMPLE: <p>Name: <input type="text" ng-model="name"></p>

DIRECTIVES 3: Bind your model data

Add the ng-bind="name" Directive to any tag. EXAMPLE:<p ng-bind="name"></p>

DIRECTIVES 4: Initialize your model data

Add ng-init="firstName='John'" Directive to any tag
